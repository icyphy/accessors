If you do not see an accessor below, then you should run the server in the accessors repository: 
   cd accessors/web/hosts/browser/test
   node testServer &
where "accessors" is the root of your repository copy, and then point your browser to this page: 
   http://localhost:8088/hosts/browser/test/testWebPage.html
If you do not see documentation, you can create the documentation as follows: 
   cd accessors/web
   ant
