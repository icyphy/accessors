Accessors

Select an accessor from the library on the left to view and interact with it.

If you do not see an accessor library on the left, then you should run the server in the accessors repository: 

   cd accessors/web/hosts/browser/test
   node testServer &
where "accessors" is the root of your repository copy, and then point your browser to this page: 
   http://localhost:8088/hosts/browser/test/testAccessorDirectory.html
If you do not see documentation for the accessors, you can create the documentation as follows: 
   cd accessors/web
   ant
