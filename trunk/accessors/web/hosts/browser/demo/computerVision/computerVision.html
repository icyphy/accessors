This demo applies image transforms to a test image. Select a transform, enter any options, and click 'react to inputs'.

Supported options:

  • {"blurSize" : 1 - 25} (blur, gaussianBlur, medianBlur)
  • {"cannyThreshold" : 10 - 150} (findContours, findEdges)
  • {"erosionSize" : 0-21} (dilate, erode)
