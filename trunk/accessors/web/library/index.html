TerraSwarm Accessor Library

The TerraSwarm accessor library is an SVN repository offering open-source code for accessors, swarmlets and host platforms.

Click on the sub-libraries in the left menu to expand them. Click on an accessor to inspect it or execute it. Try the TestAccessor to start.

Accessors are displayed using the browser host. However, not all accessors can execute under the browser host. Please see the status page to view host support for the accessors.
