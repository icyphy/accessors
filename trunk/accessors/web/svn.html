Using Subversion to update this page

This page is updated every five minutes via a cronjob run as the vc user on moog that does a svn update. The output is at svn.out.

The docs are updated by a Jenkins job, see JSDoc Continuous Integration (Not world readable)

To update these pages by hand, see How do I update workgroup pages using SVN?

Once you have an account and access, you can check out a tree with:

    svn co https://repo.eecs.berkeley.edu/svn/projects/terraswarm/accessors/trunk/accessors

Note that this svn repo is also available via anonymous access:

    svn co https://repo.eecs.berkeley.edu/svn-anon/projects/terraswarm/accessors/trunk/accessors
