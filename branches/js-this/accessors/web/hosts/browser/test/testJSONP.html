This demo shows how the JSON with padding (JSONP) technique can enable cross-domain data transfer.

Try "http://jsonplaceholder.typicode.com/posts/1?callback=?" for the URL. Some sample data will be displayed as the response.

(For some reason, react to inputs does not seem to issue the request if a default value is given).

If you do not see an accessor below, then you should run the server in the accessors repository: 
   cd accessors/web/hosts/browser/test
   node testServer
where "accessors" is the root of your repository copy, and then point your browser to this page: 
   http://localhost:8088/hosts/browser/test/testJSONP.html
If you do not see documentation, you can create the documentation as follows: 
   cd accessors/web
   ant
